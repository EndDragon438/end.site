 hello and welcome to my site! you can call me end.
 i'm building this site half as a blog half as an art portfolio. i like learning stuff, so i've got a lot of different interests anywhere from web dev to 3d art.

 HOORAY!!!!!! THE SITE IS NOW IN V1.0! i've finished implementing the tag system, and though it's only semi-automatic it's really amazing to watch!! i eventually settled on using arrays (inspired by Zonelets) to store posts, blog articles, and tags. i'm using JSON files to create new posts, cause it's a lot more human readable and kinda like a form to fill out. i'm hoping to make a post generator sometime soon so i hardly even have to touch the actual JSON file, but there's a little bit more to do first. i'm using fetch() to get the JSON files from the /posts directory, and then use that along with an .html page i copy-paste to pass into a function that generates the post layout based on the post type! i still need to implement different layouts for different post types, everything is using the 2D layout right now, but i'm really happy with where i'm at right now. beyond all the post creation stuff, i index pages on the Tag Pages by looping through the post JSON array, which lists all the posts i want displayed. it loops through fetch()ing each JSON and adding it to the Tag Page as another list item, matching the post tags against the tag provided by the Tag Page. if you want to see all i've done with the scripting, you can find it here. next, i'll be adding more layouts depending on the post type, as well as adding some of my old projects to the Creations page!!
