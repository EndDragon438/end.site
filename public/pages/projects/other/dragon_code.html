The Dragon Code

 The Dragon Code (DC2) is one of several Sig File codes that were around in the early days of the internet with the earliest (i know of at least) being the Geek Code in the early 90s. Nowadays, nearly all the Codes are extinct, except for the Furry Code which still has a website and a small amount of users. The Dragon Code is also extinct, having last been updated in Jan. 2004, and the official website went down in Oct. 2013. As far as I know, that was the end of maintainment for the Dragon Code (If anyone knows of a later maintainer, please let me know!!!). Even though the Dragon Code is pretty much extinct, it still sees some use among dragons, including myself!! And because I use the DC2, I'm trying to learn webdev, and I just happened to be bored, I decided to build a fully functional decoder for the DC2 in JS!! The Dragon Code previously had a couple decoders and encoders (listed on the official site) but only one was web-based and none of them have been archived (that I could find) also, none of them were listed as v2.6 compliant, which was the last version of the DC2. So, I present, the Dragon Code v2.6 Decoder!! It took me one sleepless week (the colour took 3 whole days), but it's done!! Technically, it doesn't support complex species (meaning species including extra species inside {}) but I plan on adding that soon (it should be pretty simple) and the official site recommends just making seperate codes anyways. You'll notice that I also have an Encoder tab, and I do plan on making an encoder in the future. The encoder is a lot more simple, mostly just a lot of dropdowns, but the main focus of this project was the decoder (the more complex one!) so I'm making this page with it still WIP.
 Also, if my decoder/encoder get enough interest, and if people request changes, I may take up maintaining the Dragon Code (if no one else is of course) and make a page for it. I'm still kinda iffy on this, but if there are actually enough dragons interested I might just go for it.

If you would like to know more about the history of the Dragon Code, or Sig File codes in general, please visit this website, the (now nonexistent) front page of the official website of the Dragon Code!
The page was maintained by Wyrm, who created v2-2.6 of the Code, until the site went down in Oct. 2013
Wyrm has confirmed to me (by email) that they are no longer maintaining the Dragon Code.

Shapechangers (DC2.~)
* ~Shapechangers
 Shapechangers can indicate the forms they change between using "/" to seperate the species. ex: DC2.~Dw/H
to indicate you shapeshift between a Western Dragon and Human form.  It's quite likely your distinct forms have different characteristics, in which case you can define them using "{}". ex: DC2.~Dw{Gf}/H{Gm}A to indicate that your Western Dragon form is female, and your Human form is male, but both are mature adults. (it is recommended to make seperate codes for your forms, as if you put it all in one it could get very complex very quickly)
 You may also like to note a preferred form, in which case you may list a species along with "~". ex: DC2.~Dw to indicate you shapeshift, but are commonly found in the form of a Western Dragon.

Modifiers
* ? Unknown - DC2.? indicates 'I have no idea what I am!'
* ^ Shaped - this is used to indicate you are mainly one form with features of another. ex: DC2.H^Dw indicates you are a human with features of a Western Dragon.
* [] Form Trapped - used to indicate when you are one species trapped in the form of another. ex: DC2.Dw[H] indicates you are a Western Dragon trapped in the form of a Human.
* + Crossed - used to indicate you are a cross between two or more species. ex: DC2.Dw+H indicates you are a cross between a Western Dragon and a Human.

* G~ Variable gender; changing (if you'd like to specify, you can put your variable genders inside "()" parentheses. ex: G~(fm) if you're variable between male and female)

Modifiers
* > In transition between two genders. ex: Gm>f for transitioning from male to female
* / Differing mental/physical genders. ex: Gm/f to mean male trapped in a female body
* "" Custom gender, if you'd like to specify something outside the Code. ex: G"agender" if you are agender. If you'd like your custom gender to be compatible with the Decoder, make sure it's all lowercase. (I'd recommend using custom, as this section of the Code is a little bit dated)
